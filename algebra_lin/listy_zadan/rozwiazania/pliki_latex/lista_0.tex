  $$5(u+w)+2w=7(u+w)+(-2)u$$
  Mnozenie przez skalar jest rozdzielne wzgledem dodawania wektorow:
  $$5u+5w+2w=7u+7w+(-2)u$$
  Dodawanie wektorow jest przemienne:
  $$5u+5w+2w=7u+(-2)u+7w$$
  Mnozenie przez wektor jest rozdzelne wzgledem dodawania skalarow:
  $$5u+(5+2)w=(7+(-2))u+7w$$
  $$5u+7w=5u+7w$$

  $$\frac1{y_1}=\frac1{y_2}$$
